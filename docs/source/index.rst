Documentation
===================================
We used the pm4py's format to create the documentation

``pm4py`` is a Python library that in our project want to extend to allow support for DCR Graph presented in:
"Declarative Event-BasedWorkflow as Distributed Dynamic Condition Response Graphs", by Thomas T. Hildebrandt and Raghava Rao Mukkamala,
published in Electronic Proceedings in Theoretical Computer Science — 2011, Volume 69, 59–73. :doi:`<10.4204/EPTCS.69.5>`_.

Examples can be found in getting started
